# 「金币 + 美女 + 歌曲」落地页视觉方案 & 模板推荐

## 一、视觉方案（基于 Deep Research）

### 1. 风格定位

**方向 A：音乐/直播平台**
- 暗色霓虹夜店风 / 沉浸式音乐播放器风
- 情绪：热闹、年轻、有节奏感、奖励感
- 参考：Spotify 深色版 + Twitch 直播间 + 游戏化积分系统

**方向 B：游戏/娱乐**
- 赛博朋克游戏风 / 炫彩卡牌游戏风
- 情绪：冒险、收集、炫耀、竞技感
- 参考：原神/王者荣耀活动页 + 音游界面 + 区块链游戏 landing

### 2. 视觉元素

| 元素 | 推荐呈现 | 注意 |
|------|---------|------|
| **金币/积分/打赏** | 3D 金色硬币、发光 token、旋转奖励徽章、粒子爆开动效、进度条闪光 | 避免像赌博筹码，做成游戏金币/打赏礼物形态 |
| **美女/主播/歌手/IP** | 2D 动漫/Live2D 主流，也可用 3D 模型；高质量人物摄影、圆形头像、半身特写、光晕边框 | 高级舞台感，避免低俗暴露 |
| **音乐/歌曲** | 声波可视化、黑胶唱片、专辑封面网格、漂浮音符、律动波形、沉浸式 3D 专辑导航 | 可做成动态背景 |
| **背景/氛围** | 深紫→深蓝渐变、霓虹光晕、粒子效果、金属质感 | 暗色底突出金色和人物 |

### 3. 配色方案

- **背景**：深紫 `#1a0b2e` → 深蓝 `#0d1b2a` 渐变
- **强调色**：金色 `#FFD700`、霓虹紫 `#b026ff`、电光蓝 `#00d4ff`
- **文字**：白色/浅灰 + 金色高亮
- **情绪关键词**：暗色、霓虹、金属、发光、律动、游戏化

### 4. 排版与动效

- **Hero 区**：全屏沉浸，左侧大标题 + CTA，右侧主播/歌手人物 + 漂浮金币
- **卡片区**：模板/主播/歌曲卡片，hover 金光描边或浮起
- **积分动效**：金币掉落、数字滚动、进度条闪光、成就徽章弹出
- **音乐动效**：背景声波随滚动变化、音符漂浮、唱片旋转

## 三、Deep Research 关键发现

**已验证的设计趋势：**
1. **人物 IP 主流是 2D 动漫 / Live2D**，同时兼容 3D 模型与主题化吉祥物
2. **音乐视觉语言** 偏向「沉浸式 3D 音乐导航」和「游戏化互动微站」
3. **紫色/紫罗兰色系** 在音乐推广落地页中被明确用作氛围色
4. **Pinterest** 是收集人物/视觉参考的首选工具，**Awwwards/Behance** 作为案例库

**未验证/需要额外调研：**
- 金币/打赏/积分类 UI 的具体视觉模式（颜色、形状、动效）在本次研究中未获得充分可靠证据
- 具体的字体组合与排版层级
- 移动端落地页上的动效降级策略
