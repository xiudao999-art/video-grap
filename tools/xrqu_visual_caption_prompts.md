# 表情包 / 短视频视觉分析提示词

用于为 `xrqu_表情` 和 `xrqu_视频` 生成内容理解的 JSONL 索引。

---

## 一、表情包分析提示词

```text
你是表情包标注员。分析这张表情包图片，输出严格JSON（不要markdown代码块、不要解释、直接输出JSON对象）：
{"content":"画面内容描述：人物/动物/动作/表情/配文，1-2句中文",
"emotions":["这张图适合表达的情感词，3-6个，如 得意/无语/愤怒/开心/幸灾乐祸/尴尬/震惊/嫌弃/撒娇"],
"scenarios":["适合使用的具体场景，2-4个，具体一些，如 别人吹牛时回复/看别人出丑时/表达无奈/吐槽对方"],
"tags":["视觉标签，3-8个，如 男孩/猫/笑脸/文字/手势/流泪"]}
```

### 输出字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `content` | string | 画面内容描述：人物/动物/动作/表情/配文，1-2 句中文 |
| `emotions` | array | 适合表达的情感词，3-6 个 |
| `scenarios` | array | 适合使用的具体场景，2-4 个 |
| `tags` | array | 视觉标签，3-8 个 |

---

## 二、短视频分析提示词

```text
你是视频素材标注员。这是一段短视频素材（表情/搞笑/转场/特效/场景）。分析后输出严格JSON
（不要markdown代码块、直接输出JSON对象，数组每项是独立的词不要用斜杠合并）：
{"content":"画面内容与动作/动效描述，1-3句，注意是视频要描述动态变化、发生了什么",
"emotions":["情感词，每项一个词，如 搞笑","震惊","无语","尴尬","得意"],
"scenarios":["适合使用的场景，每项一条，如 聊天回复搞笑瞬间","视频剪辑转场","吐槽时配图"],
"tags":["视觉或动效标签，每项一个词，如 搞笑","转场","倒计时","绿幕","特效","字幕","真人","动画"]}
```

### 输出字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `content` | string | 画面内容与动作/动效描述，1-3 句，突出视频的动态变化 |
| `emotions` | array | 情感词，每项一个独立词 |
| `scenarios` | array | 适合使用的场景，每项一条 |
| `tags` | array | 视觉或动效标签，每项一个独立词 |

---

## 使用位置

- 表情包分析结果写入：`索引/xrqu_表情/<catkey>.jsonl`
- 视频分析结果写入：`索引/xrqu_视频/<catkey>.jsonl`
- 总索引文件：`索引/_index.json`
